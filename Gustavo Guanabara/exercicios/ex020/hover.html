No Exemplo acima, usamos uma DIV com um texto fora do paragrafo e um texto dentro do paragrafo, no css vamos configurar da seguinte forma:

div > p display: none; Para indicar que queremos mexer apenas no paragrafo dentro da DIV. Display none para o paragrafo ficar invisivel.

div:hover > p display: block; Dessa forma configuramos para o paragrafo aparecer apenas quando se passar o mouse em cima. Display block vai deixar visivel.
