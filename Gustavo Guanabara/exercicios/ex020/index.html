Usamos uma "div" para selecionar e trabalhar um area da tela. Para selecionar o tamanho da "DIV" vamos usar altura e largura "height" "width" e colocar o tamanho em px.

No css as classes são representadas da seguinte forma:

# é usada para um "ID".

. é usado para uma classe.

: é usado para uma pseudo classe.

display: inline-block; vamos usar para deixar as div alinhadas em linha horizontal.

border: 1px solid black; vamos usar para criar uma borda.

div:hover usando o elemento hover aplicado a uma pseudo classe em div, consguimos configurar a div para que a configurção presente apareça apenas quando interagimos com a div, nesse caso quando passar o mouse em cima ela irá mudar de cor.
