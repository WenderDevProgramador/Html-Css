Trabalhando com listas em HTML.

Listas Ordenadas.

  1. Acordar
  2. Tomar cafe
  3. Mandar mensagem de bom dia
  4. Tomar banho e escovar os dentes
  5. Sair pra correr.

Usando a TAG OL e dentro dela a TAG LI conseguimos criar listas ordenadas em ordem.
Usando o tipo TYPE dentro de OL conseguimos determinar o tipo de sequencia da lista, se é númeral, alfabetica ou número romano.
Usando START dentro de OL conseguimos colocar em que número ou letra ira começar a lista.

Listas não ordenadas.

  • Arroz
  • Feijão
  • Macarrão
  • Lentilha

Usando a TAG UL conseguimos criar uma lista não ordenada e dentro de UL usando TYPE = "disc" para formado de bolinha, TYPE="circle" para formato de circulo e TYPE="square" para formato de quadrado.
