Testando Gradiente.

Para criar um degrader na cor de fundo. Vamos no cabeçalho em "head" adicionar "style" parametro "body" escolher "background-image: linear-gradient(to right,white,blue);" são três parametros o lado para qual o degrader ira começar e a cor inicial e a cor final.

Para fazer o degrade começar de baixo para cima ou de cima para baixo vamos usar a configuração global em css * usando asteristico, height 100% para colocar em altura de 100% da pagina. E em linear-gradient to top para começar de cima ou bottom.

Para deixar o degrader fixo usamos blackground - attachment: fixed

Em vez de escolher o lado do degrader, left ou rigth, usando -45deg podemos escolher o angulo do degrader.

Colocando % a porcentagem junto com o código da cor conseguimos colocar a porcentagem da cor sera exibida na tela mais ou menos.
