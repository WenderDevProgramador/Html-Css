a { color: blue; text-decoration: none; font-weight: bold; } Vamos usar essa configurção para mudar a cor dos links. "Font Weigth" para colocar em negrito. Para tirar o sublinhado "text-decoration: none;" usamos essa configuração.

a:visited { color: brown; } Usamos essa configuração com uma pseudo class para mudar a cor do link quando ele já foi visitado.

a:hover { color: black; text-decoration: underline; } Usamos essa configuração para quando passar o mouse em cima do link aparecer o sublinhado e ficar preto.

a:active { color: chartreuse; } Usamos essa configuração para quando clicamos em cima ele ficar verde.

a::after { content: '[link]'; text-decoration: none; font-weight: normal; color: darkgray; } Usamos essa configuração de "::" pseudo element para avisar que é um link.
