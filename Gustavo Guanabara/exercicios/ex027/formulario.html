Meu primeiro Formulario.

Sempre que criar um formulario vamos usar a Tag FORM e dentro dela usaremos input do tipo text e para criar o botão de dados usaremos input tipo submit.

Para desligar ou ligar o auto preenchimento dos campos usamos form autocomplete="off"

Os dados do formulario sera enviado para um arquivo de linguagem de programação. Atraves de action dentro da tag form.

Para fazer a ligação entre a caixa e o parametro vamos fazer uma etiqueta, será feito atraves do name e id usaremos a tag label e no parametro for colocamos o id.

Existe dois metodos para envio de formularios o parametro method="post" e method="GET", o metodo usando o paramentro get em form ira aparecer os dados do formulario na url, o metodo usando o paramentro post em form ira tirar os dados enviados da url mas isso não quer dizer que estara seguro os seus dados. Para proteger realmete os dados precisamos criptografar e usar https.

So pode enviar dados atraves do GET até 3 mil bytes (Conseguimos enviar mais dados atraves do POST). Também não devemos usar para dados sensiveis como senhas e outros.
