form é a tag usada para formularios
action nome do arquivo aonde os dados seram enviados
metho é a forma como os dados seram enviados
autocomplete é usado para que tenha o prenchimento automatico dentro da caixa.

label é usado para criar um vinculo entre a caixa input e oque o formulario pede
for usamos para colocar o id do input para fazer a ligação entre os dois.



input type="password" name="senha" id="isenha" required minlength="8" maxlength="16" size="16" placeholder="senha min 8 letras" autocomplete="current-password"

input para criamos a caixa para recolher os dados
type será o tipo de dados que sera recolhido ex: text , passaword , number etc
name e id são a identificação do input
required para não deixar enviar os dados vazios
minlength e maxlength aqui será definido o maximo e o minimo de caracteres permitidos no formulario
placeholder ira da uma dica dentro da caixa de input
autocomplete será para o navegador completar a caixa de input
size tamanho da caixa de acordo com a quantidade de caracteres

input type="reset" value="Limpar"
