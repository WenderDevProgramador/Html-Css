Principais formatações.

Negrito/Destaque

Nessa frase temos um termo em negrito usando a tag B.

Nessa frase temos um termo em destaque usando a tag STRONG.

Italico/Enfase

Nessa frase temos um termo em Italico usando a tag I (não semantica).

Nessa frase temos um termo em italico usando a tag EM(com enfase e semantica).

Texto marcado.

Podemos usar a taga MARK. Para marcar um texto.

Texto grande / Texto pequeno.

Para se deixar um texto com a letra grande usa a tag BIG. Porém essa tag já foi excluida uma vez que se não ver utilidade já que o aumento de letra e texto deve ser feito no CSS#

Para se deixar um texto com a letra pequena usa a tag SMALL.

Texto deletado.

Podemos marcar um texto como excluido para mostrar que ele deve ser lido mas não considerado.Usa a TAG DEL.

Texto incluido.

Podemos marcar um texto como incluido para da enfase que ele foi adicionado depois.Usando a TAG INS.

Texto sobre escrito.

Para torna um texto sobre escrito usa a tag SUP. Exemplo X20+3

Texto subi escrito.

Para torna um texto subi escrito usa se a TAG SUB. Exemplo h2o.
