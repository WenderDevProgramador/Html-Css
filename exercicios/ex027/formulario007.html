Formularios Exemplo


select name="est" id="iest" Select é uma tag como o input usamos para colocar uma lista pronta aonde o usuario ira fazer a escolha.

option value="" selected É apoção de mensagem que vamos deixar dentro da caixa select.

optgroup label="Região Sudeste" Essa tag vamos usar dentro de select com os option para separar a seleção em grupos.

option value="PE" É a opção que vamos deixar para o úsuario.

input type="text" name="prof" id="iprof" list="lstprof" Essa tag vamos usar no input text para que o úsuario digite um valor mas que tambem tenha uma opção de escolha. "list" precisa ser colocado para que façamos uma interligação com o datalist.

datalist id="lstprof" Vamos colocar a opção para o usuario escolher, dentro de datalist vem o option para colocar as opções não podemos nos esquecer do id para interligar com o input.
