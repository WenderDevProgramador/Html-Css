Usando O Google Fonts.

Podemos usar o Google Fonts para pegar mais opção de fontes, dessa forma já conseguimos pegar o link e o código css pronto para usar na nossa pagina html e css3.

Usando Fontes Externas Do Cliente.

As configurações do google fonts para tirar duvidas estão em css local e da dafont.com vou deixar em inline para separar as configurações e não precisar gerar outro arquivo.

Um site bom para pesquisar fontes externas caso não tenha no google fonts é o dafont.com .

Segue a baixo o código para usar fontes esternas baixadas para a mesma pasta do arquivo da pagina: 

    @font-face {
        font-family: 'Magical';
        src: url('Magical Celebration.ttf')format('truetype');
        font-weight: normal;
        font-style: normal;
    }

Cada tipo de arquivo terá um tipo diferente, no caso do tipo ttf será format('truetype') e no caso do otf será format('opentype').
