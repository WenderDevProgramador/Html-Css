Aprendendo Html.

Anotação Importante.

Para criar um seletor para se por uma configuração diferente vamos até o "elemento" que queremos mudar e vamos diferencialo colacando ID "diferenciado" dessa forma conseguimos diferenciar um h1 de outro h1 por exemplo, na pagina de configuração css esse ID sera # essa será a forma de diferencialo a na configuração colocando # mais o nome.

Lembrando que o seletor ID em um elemento só pode ser usado em um unico elemento diferenciado, para diferenciar varios elementos vamos usar o seletor "CLASS". De acordo coma regra da w3c só pode ter um ID dentro de um documento.

Para o seletor CLASS conseguimos diferenciar a configuração de varios elementos criando uma classe, conseguimos colocar essa classe tambem junto como ID, o seletro CLASS em CSS sera conficgurado como um ponto "." . Para classificar um trexo do texto como class vamos usar a TAG "SPAN" a principio ela não farar nada até que você coloque uma clsse nela. Um elemento pode ter mais de uma class só adicionar o nome classe pode ter varios nomes.
