Fontes Em CSS.

Anotação importante.

Usar para tamanho de letra sempre PX ou EM! "EM" é mais usada para trabalhar com porcentagem é melhor para ultilizar com diferentes tamanhos de telas.

Font - size = Tamanho.

Font - weight = Peso da fonte.

Font - width = Largura.

Font - height = Altura.
