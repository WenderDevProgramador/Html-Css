Diferentes tamanhos de imagens em HTML

Para realizamos a troca de tamanho de imagem de acordo com o tamanho da tela vamos usar a seguinte configuração. TAG "PICTURE"Dentro dela vamos usar a TAG IMG a imagem com o tamanho inicical, e também a TAG SOURCEmedia="(max-width:1050px )" colocando sempre 50 px a mais do tamanho da imagem,srcset="imagens/media.png" coloca-se a imagem que vai subistituir caso o tamnho da tela tenha mudandça, type=" image/png" e o tipo da imagem. Sempre respeitar a ordem do maior para o menor ou do menor para o menor.
